# Exquisite Corpses
[What is an exquisite corpse?](https://en.wikipedia.org/wiki/Exquisite_corpse)

A small installation consisting of torsos, legs, and your face

## How to Start
```
yarn
yarn start
```

## Installation
The program should be projected in a dark room against a wall.
